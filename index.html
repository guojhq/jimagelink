FeaturesTop

此插件是模拟Yupoo的缩略图的功能，当鼠标经过图片时，显示出一个小图标，点击可以看大图，再结合类似Lightwindow这种类似的插件实现图片显示效果。
本页的演示效果使用了Shadowbox

JS:

						
$("body").ready(function(){

	/*
		there is the jImageLink code
	 */
	$("a.imgLink").jImageLink({		
		iconClassName : "linkIcon",
		iconRel : "shadowbox",
		iconWidth : 14,
		iconHeight : 14
	});		
	
	/* use shadowbox */
	Shadowbox.init({
		resizeLgImages:     true,
		displayNav:         false,
		handleUnsupported:  'remove',
		keysClose:          ['c', 27] // c or esc
	});

});

CSS:

a.linkIcon {
	display: block;
	background: url(images/lightbox.gif) no-repeat;
	width: 14px;
	height: 14px;
}

img {border:none;}
